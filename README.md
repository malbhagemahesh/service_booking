# 🚀 Service Booking Platform

A full-stack web application for booking services like veterinary, grooming, and more.

---

## 🛠 Tech Stack

* Frontend: React + Tailwind CSS
* Backend: FastAPI
* Database: MongoDB

---

## ⚙️ Features

* User Registration & Login
* Service Listing & Booking
* Provider Dashboard
* Responsive UI

---

## 📦 Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🌐 API Endpoints

* POST /auth/register
* POST /auth/login
* GET /services
* POST /bookings
